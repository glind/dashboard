# Support

## 30-Day Installation & Configuration Support

All installations of Personal Dashboard include **30 days of installation and configuration support** from the date of first deployment.

### What's Covered

✅ **Installation Assistance**
- Help with Docker deployment setup
- GitHub Pages configuration
- Environment variable configuration
- Database initialization
- OAuth setup for Google, GitHub, TickTick

✅ **Configuration Support**
- API key configuration
- Collector setup (email, calendar, tasks, etc.)
- AI provider configuration (Ollama, OpenAI, Gemini)
- Troubleshooting startup issues

✅ **Basic Usage Guidance**
- Dashboard navigation
- Feature explanations
- Settings and customization

### What's NOT Covered

❌ Custom feature development
❌ Third-party API issues (Google, GitHub, TickTick, etc.)
❌ Infrastructure/hosting issues
❌ Code modifications or extensions
❌ Performance optimization beyond documented best practices

### How to Get Help

1. **Check Documentation First**
   - Review `devdocs/SETUP.md` for installation
   - Check `devdocs/OPERATIONS.md` for deployment
   - See `devdocs/REFERENCE.md` for API details

2. **Search Existing Issues**
   - Someone may have already solved your problem
   - Search closed issues too

3. **Open a Support Request**
   - Use GitHub Issues template
   - Include: OS, Python version, deployment target, error logs
   - Provide steps to reproduce

4. **For Urgent Issues (Labs Customers Only)**
   - Email support@buildly.io with "[URGENT]" in subject
   - Include your Labs customer ID
